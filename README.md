# RESUME-PARSER
# 🪷 ResumeAI — Smart HR Intelligence

AI-powered resume parsing web application designed specifically for **Indian HR workflows**, enabling intelligent candidate analysis, ATS scoring, and hiring insights.

---

## 🚀 Overview

**ResumeAI** is a front-end web app that uses advanced AI to:

- Parse resumes (PDF, images, or text)
- Extract structured candidate data
- Evaluate skills, experience, and education
- Generate hiring insights and recommendations
- Provide ATS (Applicant Tracking System) compatibility scoring

Built with a modern UI and tailored for Indian hiring patterns, it delivers **quick, structured, and actionable candidate intelligence**.

---

## ✨ Features

### 📄 Resume Input Options
- Upload formats:
  - PDF
  - Images (JPG, PNG, WEBP)
  - TXT files
- Or paste resume text directly

---

### 🧠 AI-Powered Parsing
- Extracts:
  - Personal details
  - Work experience
  - Education
  - Skills (technical, soft, domain)
  - Certifications & languages

---

### 📊 Candidate Scorecard
Evaluates:
- Technical Depth
- Communication Skills
- Leadership
- Education Quality
- Career Growth
- Overall Fit Score

---

### 🤖 ATS Compatibility Check
- Resume scoring (0–100)
- Feedback for improvement
- Helps optimize resumes for recruiter systems

---

### 💡 AI Intelligence Report
- Summary of candidate profile
- Strengths & gaps
- Hiring verdict (Strong Hire / Good Fit / Needs Improvement)
- Suggested job roles
- Estimated salary range
- Notice period insights

---

### 🎨 UI Highlights
- Elegant Indian-themed design (Saffron, Green, Gold palette)
- Smooth animations and transitions
- Fully responsive layout
- Print-ready report view

---

### 📥 Export & Actions
- Download full report as `.txt`
- Copy summary to clipboard
- Print-friendly layout

## ⚙️ How It Works

1. User uploads or pastes resume
2. App processes file (PDF/Image/Text)
3. Sends structured request to AI API
4. Receives JSON response
5. Renders:
   - Profile
   - Skills
   - Experience
   - Scores
   - Insights

---

## 🔐 Important Notes

### ⚠️ API Security
- API key is currently used in frontend calls
- **Not safe for production**

👉 Recommended:
- Move API calls to a backend server (Node.js / Python)
- Store API keys securely using environment variables
